# Email Diagnostics

## Scenario

A user reported that Outlook was running slowly and emails were delayed.

Email diagnostics were performed to identify possible mailbox or connectivity issues.

---

## Troubleshooting Steps

1. Logged into Microsoft 365 Admin Center.
2. Opened **Support and Diagnostics**.
3. Selected **Run Tests: Outlook Connectivity**.
4. Performed mailbox diagnostics.
5. Checked mailbox storage limits.
6. Reviewed service health dashboard.
7. Verified there were no Microsoft service outages.
8. Restarted Outlook after diagnostics completed.

---

## Result

No service issues were detected. Outlook performance improved after restarting the application.
